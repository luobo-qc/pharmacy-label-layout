# 药房药品标签排版网页程序设计

## 目标

设计并实现一个浏览器网页程序，用于手动录入患者用药信息，生成 `60mm x 40mm` 药品标签预览，并通过浏览器打印到热敏或标签打印机。

第一版聚焦一个明确流程：药房工作人员填写患者和药品信息，查看实时标签预览，调整少量模板选项，然后打印单张标签。

## 第一版范围

包含：

- 单页网页应用。
- 手动录入标签数据。
- `60mm x 40mm` 标签实时预览。
- 通过浏览器调用打印。
- 简单模板设置。
- 使用浏览器本地存储保存模板设置、常用项和最近打印记录。

不包含：

- 登录和用户权限。
- 服务端数据库。
- 多设备同步。
- HIS 或药房系统接口。
- 拖拽式模板编辑器。
- 处方单打印。
- 货架价签打印。
- A4 多标签排版。

## 推荐方案

采用纯前端网页应用，并使用浏览器本地存储。

这个方案适合第一版：部署简单，药房前台电脑用浏览器打开即可使用，不需要后端服务，同时还能记住模板设置、常用药品、常用用法用量和最近打印记录。

## 使用流程

1. 用户在浏览器中打开网页应用。
2. 用户填写患者和药品信息。
3. 标签预览实时更新。
4. 用户按需调整模板设置。
5. 用户点击“打印标签”。
6. 程序检查必填字段。
7. 浏览器打开打印窗口。
8. 用户选择标签打印机并打印。
9. 触发打印后，程序把常用项和最近打印记录保存到浏览器本地。

## 页面布局

桌面端使用三栏布局：

- 左侧：标签录入表单。
- 中间：`60mm x 40mm` 实时标签预览。
- 右侧：模板设置、常用项、最近打印记录。

窄屏时改为上下布局：

- 表单。
- 预览。
- 模板设置。
- 常用项和最近打印记录。

## 标签字段

| 字段 | 是否必填 | 规则 |
| --- | --- | --- |
| 患者姓名 | 是 | 为空时禁止打印。 |
| 药品名称 | 是 | 标签中最大、最醒目的内容。 |
| 规格 | 否 | 示例：`0.5g x 24片`。 |
| 数量 | 否 | 示例：`1盒`、`7袋`。 |
| 用法用量 | 是 | 示例：`每日3次，每次1片，饭后服用`。 |
| 注意事项 | 否 | 可通过模板设置隐藏。 |
| 打印日期 | 是 | 默认当天，可手动修改。 |

## 标签模板

标签尺寸固定为 `60mm x 40mm`。

默认视觉层级：

1. 药品名称。
2. 用法用量。
3. 患者姓名。
4. 规格、数量、注意事项、打印日期。

模板设置：

- 基础字号。
- 药品名称是否加粗。
- 是否显示注意事项。
- 标签内边距，可在 `1mm` 到 `5mm` 之间调整。

第一版使用结构固定、少量可调的模板。字段不能自由拖拽，也不能任意改变位置。

## 本地数据

第一版使用浏览器 `localStorage`。

保存内容：

- 模板设置。
- 常用药品名称。
- 常用规格。
- 常用用法用量。
- 最近打印记录。

数量限制：

- 每类常用项最多保存 50 条。
- 最近打印记录最多保存 100 条。

超过限制时删除最旧记录，避免本地数据无限增长。

未来如果需要大量记录、搜索、多用户或多设备同步，再升级到 `IndexedDB` 或后端数据库。

## 交互规则

- 表单输入变化时，预览实时更新。
- 点击“打印标签”前检查必填字段。
- “新建标签”清空患者姓名、药品名称、规格、数量、用法用量、注意事项。
- “新建标签”保留模板设置。
- 最近打印记录可一键回填到表单，用于重打。
- 常用项可点击选择，用于减少重复输入。
- 注意事项被隐藏时，不删除已经输入的注意事项文本。

## 打印行为

打印使用浏览器打印能力和专门的打印样式。

打印样式需要：

- 隐藏应用控制区。
- 只打印标签预览区域。
- 设置页面尺寸为 `60mm x 40mm`。
- 移除页面默认边距。

打印 CSS 方向：

```css
@page {
  size: 60mm 40mm;
  margin: 0;
}
```

程序无法完全控制不同打印机驱动造成的物理偏移。模板中的内边距设置用于让用户微调打印位置。

## 错误处理

打印前需要给出清晰提示：

- 患者姓名为空。
- 药品名称为空。
- 用法用量为空。
- 打印日期为空。

长文本处理规则：

- 预览区显示内容是否超出标签。
- 药品名称和用法用量可自动缩小到基础字号的 `80%`。
- 缩小后仍然放不下时，打印前提示用户修改内容。
- 第一版不自动截断药品名称和用法用量。

本地存储失败时，不阻止标签生成和打印。程序显示非阻塞提示，告知用户设置或历史记录未保存。

## 测试策略

核心测试覆盖：

- 必填字段校验。
- 表单状态生成预览。
- 模板设置保存和恢复。
- 常用项 50 条上限。
- 最近打印记录 100 条上限。
- `60mm x 40mm` 打印样式。
- 长文本提示。
- 最近打印记录回填。

人工验收覆盖：

- 桌面宽屏预览。
- 窄屏预览。
- 浏览器打印预览中的 `60mm x 40mm` 页面尺寸。
- 常见药房标签内容在预览中清晰可读。
